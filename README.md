<h1>React Router</h1>

<p> Lesson plan proposal about routing in React </p>

<p>Proposta de plano de aula sobre roteamento em React</p>
<h2>Objectives (Objetivos)</h2>
<ul>
<li>Understand the concept of routing in web applications (Compreender o conceito de roteamento em aplicações web)</li>
<li>Learn how to implement routing in React using React Router (Aprender a implementar roteamento em React usando React Router)</li>
<li>Create multi-page applications with navigation (Criar aplicações multi-página com navegação)</li>
</ul>

<h2>Topics Covered (Tópicos Abordados)</h2>
<ul>
<li>Introduction to Routing (Introdução ao Roteamento)</li>
<li>Setting up React Router (Configuração do React Router)</li>
<li>Defining Routes (Definindo Rotas)</li>
<li>Navigation with Links (Navegação com Links)</li>
<li>Handling Route Parameters (Manipulação de Parâmetros de Rota)</li>
<li>Nested Routes (Rotas Aninhadas)</li>
<li>Redirects and Navigation Guards (Redirecionamentos e Guardas de Navegação)</li>
</ul>
